Types of Trigger

When you define a trigger, you can specify the number of times the trigger action is to be excuted. once for every row effected by the triggering satement (row-level trigger) or once for the triggering statement, no matter how many rows it affects (statement-level trigger).

Depending upon, when a trigger is fired, it may be classified as :

(i) statement-level trigger/ table lavel trigger
(ii) row-level trigger
(iii) Before trigger
(iv) After trigger


(i) statement-level trigger/ table lavel trigger : A statement-level trigger is fired once on behalf of the triggering statement regardless of the number of rows in the table that the triggering statement affects.

  • Example of statement-level trigger/ table lavel trigger

    • Select * from product;

    create or replace trigger table_level
    before update
    on product
    begin
    Dbms_output.put_line('The product table values is updated');
    end;




    update product set p_name='frooti', price=56 where p_id=103;




    (ii) row-level trigger : A row levell trigger is fired each time the table is affected by the triggering statement, i.e it is fired once for each row that is effected bye the triggering statement.

  • Example of row-level trigger

    • create or replace trigger row_level
    before delete on product
    for each row
    begin
    dbms_output.put_line('The value is deleted ');
    end;





    delete from product where p_id=101 or P_id=102;




    (iii) Before trigger : when before a trigger you can specify the trigger timing. That is you can specify whether the trigger action is to be excuted before or after the triggering statement.

  • Example of Before trigger

    • create or replace trigger before_trigger
    before insert on product
    for each row
    begin
    dbms_output.put_line('The new product name is : ' ||:new.P_name);
    end;





    insert into product(p_name) values('frooti');
    insert into product(p_name) values('kitkat');




    (iv) After trigger : after trigge are fired after the triggering action is completd.
