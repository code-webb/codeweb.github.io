Commit Command

COMMIT : Commit is used for saving the DML (data manipulation language) Transaction. By Default in mysql the Autocommit is enable that is if we executed any Transaction then that Transaction is Autocommit save so to disabled the Autocommit the command is :

SET AUTOCOMMIT ON : Enable autocommit feature (Command : SET AUTOCOMMIT = 0;)
SET AUTOCOMMIT OFF : Is the Default and disables the autocommit commiting (Command : SET AUTOCOMMIT = 1;)

  • Syntax Of Commit command

    • Mysql> COMMIT;

  • Example of Commit command

    • select * from city;
    insert into city values(3,"Chandigarh");
    select * from city;
    COMMIT;
    select * from city;
    ROLLBACK;
    select * from city;
