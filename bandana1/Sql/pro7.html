Select Database

Select Database : Suppose database users and administrators want to perform some operations on tables, views, and indexes on the specific existing database in SQL. Firstly, they have to select the database on which they want to run the database queries.

Any database user and administrator can easily select the particular database from the current database server using the USE statement in SQL.

  • Syntax Of Select database

    • Mysql> USE DATABASE_NAME;
