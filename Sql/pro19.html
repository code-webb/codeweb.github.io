Order By Clause

The SQL ORDER BY clause is used to sort the data in ascending or descending order, based on one or more columns. Some databases sort the query results in an ascending order by default. The ORDER BY clause in SQL is used for sorting the records of the database tables.

  • Syntax Of Order By Clause

    • Mysql> SELECT column_name
    FROM table_name
    ORDER BY column1 [ASC] | [DESC];
