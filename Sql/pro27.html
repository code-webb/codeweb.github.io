Delete Single row

The SQL DELETE statement is used to delete rows from a table. Generally DELETE statement removes one or more records from a table.

Deleting a single row in a table! Notice the WHERE clause in the DELETE statement. The WHERE clause specifies which record(s) should be deleted. If you the WHERE clause, all records in the table will be deleted!

  • Syntax Of Delete Single row

    • Mysql> DELETE FROM table_name WHERE CONDITION ;

  • Example of Delete Single row


    • delete from student where rollno = 1;
