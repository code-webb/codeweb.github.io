Date function

Dates are complicated for newbies, since while working with database, the format of the date in table must be matched with the input date in order to insert. In various scenarios instead of date, datetime (time is also involved with date) is used.


There are default date functions are :

1. NOW()
2. CURDATE()
3. CURTIME()


1. NOW(): This function is used to returns the current date and time.

  • Syntax of NOW() Function

    • Mysql> SELECT NOW() FROM dual;

  • Example of NOW() Function

    • SELECT NOW() FROM dual;




    2. CURDATE(): This function is used to returns the current date.

  • Syntax of CURDATE() Function

    • Mysql> SELECT CURDATE() FROM dual;

  • Example of CURDATE() Function

    • SELECT CURDATE() FROM dual;




    3. CURTIME(): This function is used to returns the current time.

  • Syntax of CURTIME() Function

    • Mysql> SELECT CURTIME() FROM dual;

  • Example of CURTIME() Function

    • SELECT CURTIME() FROM dual;
